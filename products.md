# Products

## Definitions

[UEM (Unified Endpoint Management)](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#UEM) is a class of software tools that provide a single interface to manage various types of user devices, with a focus on user experience, security, and compliance. It represents an evolution of RMM tools.

[RMM (Remote Monitoring and Management)](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#RMM---class-of-tools) is a class of tools used by Managed Service Providers (MSPs) to monitor and manage client networks and devices remotely.

[PSA (Professional Services Automation)](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#PSA-%5Bgeneric%5D) is software for automating project and billing management for companies. Used in the RMM, N-central, ServiceDesk and MSP Manager products. RMMs integrate with PSA solutions.

## N-central, N-sight and Ecoverse (UEMs/RMMs)

* **N-central** - is an <ins>on-prem RMM</ins> solution provided by N-able.
* **N-sight** - is a <ins>cloud RMM</ins> solution provided by N-able.
* **Ecoverse** - is a <ins>cloud RMM</ins> solution resulting of project <ins>Fusion</ins>, which is a fusion of N-central and N-sight.

Features of Ecoverse are being integrated into N-central via the [shim-app](https://github.com/nable-nc/shim-app).

The shim-app is an iframe-based Angular application that interacts with N-Central using post messages, enabling access to Ecoverse features.

## ServiceDesk and MSP Manager (PSA Solutions)

[ServiceDesk](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#SD) and [MSP Manager](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#MSPM) are N-able Professional Services Automation (PSA) solutions, with MSP Manager being a specific PSA system and ServiceDesk being the name for the integrated service desk capabilities within N-able's offerings.

Both are used to manage and resolve customer issues efficiently by providing ticketing, billing, and scheduling features.

* **MSP Manager**: This is a dedicated, cloud-based PSA system from N-able that offers comprehensive features like a ticketing system, scheduling, and billing.
* **ServiceDesk**: This refers to the integrated service desk functionality that is built into N-able's solutions, such as MSP Manager. It handles help desk ticketing, which can be automated and integrated with other tools like N-sight RMM.
* **Integrated solutions**: N-able offers these tools to provide a consolidated platform for managed service providers (MSPs) to manage their help desk, billing, and scheduling all in one place.

## Automation Manager

[Automation Manager](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#Automation-Manager) is used in the RMMs (N-central) products. Provides a simple drag and drop interface to create an Automated Task to be run through N-central. Based on a third-party product called **Gridvision** that was purchased by both LogicNow and Solarwinds.

[Gridvision](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#Gridvision) is a third-party product that was acquired by Solarwinds MSP (and is owned by N-able) to provide the drag-and-drop capability for the Automation Manager.

[AMP](https://n-able.atlassian.net/wiki/spaces/MEO/pages/45059855926/MSP+Glossary+of+Acronyms+and+Abbreviations#AMP) is used in the RMM (N-central) products.
* **Automation Manager Policy**: A type of file executed by the ScriptRunner.
* **Automation Manager Profile**: An individual Task created using Automation Manager.
